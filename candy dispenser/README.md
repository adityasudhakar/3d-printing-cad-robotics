# Candy Dispenser

This folder contains a recreated candy dispenser CAD set made with `OpenSCAD`, `Codex`, and `Claude`.

The reference was the original **Candy dispenser :)** by `iPadNanito` on Thingiverse:

- https://www.thingiverse.com/thing:166380

Important: the CAD files here are not copies of the original Thingiverse CAD/STL files. We did not import or reuse those files. The workflow here was to feed a picture from the original listing into AI tools and iteratively recreate the model geometry from scratch in `OpenSCAD`.

The original design credit stays with `iPadNanito`. This folder is the AI-assisted recreation work and the print exports generated from that recreation.

## Files

- `codex-sketch-to-code.scad`: current editable OpenSCAD source
- `codex-sketch-to-code-base.stl`: base mesh export
- `codex-sketch-to-code-body.stl`: body mesh export
- `codex-sketch-to-code-roller-holder-short.stl`: roller-holder-short body section mesh export
- `codex-sketch-to-code-cover.stl`: cover mesh export
- `codex-sketch-to-code-roller.stl`: roller mesh export
- `codex-sketch-to-code-base.gcode`: sliced base print
- `codex-sketch-to-code-body.gcode`: sliced body print
- `codex-sketch-to-code-cover.gcode`: sliced cover print
- `codex-sketch-to-code-roller.gcode`: sliced roller print
- `codex-sketch-to-code-roller-holder-short.gcode`: sliced roller-holder-short body section print
